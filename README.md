# ⚖️ AI Contract Intelligence System

## 🚀 Overview
AI Contract Intelligence System is an AI-powered solution that analyzes legal contracts, detects risky clauses, and generates insights using a combination of RAG (Retrieval-Augmented Generation) and Multi-Agent architecture.

This project simulates how AI can assist legal teams in contract review, risk detection, and compliance analysis.

---

## 💡 Key Features

- 📄 Contract parsing and clause extraction
- ⚠️ Risk detection engine (High / Medium / Low)
- 🔎 RAG-based legal knowledge retrieval (FAISS)
- 🤖 Multi-Agent system for contract analysis
- 📊 Interactive dashboard using Streamlit

---

## 🏗️ Architecture

User Input → Parser Agent → Risk Detection → RAG Retrieval → Multi-Agent Decision → Dashboard
